What is a Loss Free prize pool?

It's never been done before, but it's actually pretty simple. We are just the first. It was one of those ideas. "Why hasn't anyone created this yet??" The lottery is based on a staking system. It chooses the highest return stake and the lottery pool is invested for a specified amount of time. The amount made from interest depends on how big the lottery pool gets. Imagine how big the gains would be if it got to a few million! At the end of the time period the original ticket purchase is returned to all the purchasers and the profit made from the lottery pool goes to the winner.
